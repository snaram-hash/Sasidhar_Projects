# Future Roadmap

See the root `ROADMAP.md` for a high-level overview. Technical milestones include:

## v2.0 (Q3 2026)
- Integration of Tesseract OCR for unstructured, handwritten financial processing.
- Fuzzy string matching algorithms to automatically group bank transactions by major suppliers/buyers.

## v3.0 (Q1 2027)
- Migration from SQLite to PostgreSQL.
- Dockerization of the application and deployment via Kubernetes.
- Implementation of an LLM agent to automatically draft the "Credit Officer Narrative" section of the CAM based on the numerical risk alerts.
